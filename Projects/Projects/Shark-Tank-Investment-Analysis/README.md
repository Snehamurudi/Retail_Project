# 📊 Shark Tank India – Startup Investment Analysis

## 🚀 Project Overview

This project analyzes startup investment trends using the **Shark Tank India dataset**. The goal is to uncover patterns in funding decisions, investor behavior, and founder success factors using data analysis and visualization techniques.

## 🎯 Objectives

* Analyze industry-wise investment trends
* Identify factors influencing funding success
* Study founder characteristics and gender distribution
* Compare investments made by different sharks
* Build an interactive dashboard for insights

## 🛠️ Tools & Technologies Used

* **Python (Pandas, Matplotlib, Seaborn)** – Data cleaning & analysis
* **Tableau** – Data visualization & dashboard creation
* **Excel** – Initial data exploration

## 📊 Key Analysis Performed

* Data cleaning and preprocessing
* Industry-wise funding analysis
* Funded vs Not Funded startup classification
* Top funded startups identification
* Gender distribution analysis
* Shark-wise investment comparison

## 📈 Key Insights

* Food, Technology, and Lifestyle industries received the highest funding
* A majority of startups secured investment deals
* Male founders outnumber female founders in pitches
* Startups with multiple investors received higher funding
* Certain sharks (like Aman Gupta, Namita Thapar) are highly active investors

## ▶️ How to Run the Project

1. Clone the repository
2. Open `analysis.ipynb` in Jupyter Notebook
3. Run all cells to perform analysis
4. Open `.twbx` file in Tableau to view dashboard
