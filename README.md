# Retail Business Performance & Profitability Analysis
## 📌 Objective
Analyze retail sales data to identify profit trends, loss-making products, and improve business decisions.

## 🛠 Tools Used
* SQL (Data Analysis)
* Python (Pandas, Matplotlib)
* Tableau (Dashboard Visualization)

## 📊 Key Insights
* Technology category is most profitable
* Furniture category has low profit
* Tables sub-category shows losses
* Discounts negatively impact profit
* Inventory days have weak correlation with profit

## 📈 Dashboard Features
* Profit by Category
* Loss-making Products
* Sales Trend (Monthly)
* Region Filter

## 💡 Recommendations
* Reduce discounts on loss products
* Focus on high-profit categories
* Optimize inventory management
* Improve pricing strategies
